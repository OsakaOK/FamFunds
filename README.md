# Sumly
A personal expense tracker. Log transactions, categorize them, and see where your money goes month-to-month.

# Features (planned)
- JWT-based authentication; per-user data isolation
- Add, edit, delete, and filter transactions
- Categories, accounts, and monthly aggregation
- Responsive React UI with monthly spend chart

# Tech Stack
- Backend: Java 17, Spring Boot 3.x, Spring Security, JWT
- Database: PostgreSQL
- Frontend: React, Recharts
- Local dev: Docker Compose
